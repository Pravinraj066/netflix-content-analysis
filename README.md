# 🎬 Netflix Content Analysis Dashboard

> End-to-End Data Analytics Project using **Python, MySQL, and Power BI**

---

## 📌 Project Overview

This project is an end-to-end Data Analytics solution that analyzes Netflix's movies and TV shows dataset to uncover meaningful business insights.

The project demonstrates the complete analytics workflow, including:

- Data Cleaning & Preprocessing using Python
- Business Analysis using MySQL
- Interactive Dashboard Development using Power BI

The final dashboard provides insights into Netflix's content library through KPI cards, interactive charts, and dynamic filters.

# 💡 Key Insights

- Movies account for the majority of Netflix's content library.
- The United States contributes the highest number of titles.
- TV-MA is the most common content rating.
- Netflix experienced significant content growth after 2015.
- Drama and Documentary are among the most popular genres.
